FALCON TENTS - DEPLOYMENT PACKAGE
================================

This is a pre-built Next.js standalone deployment. Do NOT run npm install or npm run build.

TO RUN:
-------
  node server.js

OPTIONAL:
---------
  - Set PORT=3000 (default) via environment variable
  - Set HOSTNAME=0.0.0.0 to listen on all interfaces
  - Use PM2: pm2 start server.js --name falcon-tents

REQUIREMENTS:
-------------
  - Node.js 18 or higher
  - Runs on Hostinger VPS, cPanel Node.js, or any Node.js hosting
